An opinionated react Progressive Web App repository

26 February 2017 on Javascript, Best Practices, Latest Trends

Over the past week or so I have been working on understanding the ins and outs of Progressive Web Apps. In this endeavour, I decided that I would create my very own (very opinionated) repository full of all the tools that I need to create a quick and stable static application.

The repository makes use of the following:

  • React
  • Mocha
  • Nightwatch.js
  • Webpack 2
  • Flow
  • ESLint

Please see the link below for the full repository, if you are reading this and fancy helping out on some of the outstanding issues, please create an issue on GitHub, and we can discuss.

https://github.com/simonfl3tcher/react-progressive-web-app
